# API Security

## Description
This application for study some OWASP10-2023 

### Features
 * JWT for authentication,authorization each endpoint
 * HTTPS
 * Prevent Cross-site-request forgery (CSRF)
 * Prevent SQL Injection
 * prevent timing attack 

### Technologies Used
 * Node.js
 * Eexpress
 * SQL
  

  

## Installation
```
git clone 
cd project
npm install
```

## Start development
```
npm start
```
